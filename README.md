# Next.js Queue Frontend (Final)

Simple, polished Next.js App Router frontend for an on-site queue system.

Pages:
- / (home)
- /pelanggan (customer ticket)
- /petugas (staff panel)
- /display (big display)

How to run:
1. npm install
2. npm run dev
3. Open http://192.168.1.14:3000

Notes:
- This demo uses localStorage for simplicity. For production, add a backend + WebSocket & printer integration.
